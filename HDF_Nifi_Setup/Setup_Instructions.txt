Apache Nifi Sandbox in HDP setup
----------------------------
https://www.cloudera.com/downloads/hortonworks-sandbox/hdf.html

register and download Hortonworks HDF

Choose Installation Type> Virtualbox

in Oracle Vmbox: File> Import Appliance > add the HDF_3.1.1_virtualbox_180626 ova file
--Start the machine
--wait for "extractind & loading layer" to complete
--Access sandbox via http://localhost:1080/
--Access quicklinks > Advanced HDF
--Access Dashboard to look at all services within HDF
